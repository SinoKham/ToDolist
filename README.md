# SinoDo - Персональный таск-менеджер
![Интерфейс SinoDo](./screenshots/sinodoskr.png)
Современное веб-приложение для управления задачами, вдохновлённое Microsoft To Do и другими productivity-инструментами. Полностью работает на чистом JavaScript, без фреймворков. Создано в учебных целях для глубокого понимания DOM, событий, асинхронных запросов и LocalStorage.

## Как запустить
Просто открой `index.html` в браузере - никаких серверов и зависимостей не требуется.

## Возможности
- **CRUD задач** - создание, просмотр, редактирование и удаление.
- **Внешний API** - профиль пользователя (аватар, имя) загружается с randomuser.me.
- **Валидация форм** - проверка на пустоту, некорректную дату и дедлайн в прошлом. Ошибки при отправке и потере фокуса.
- **Поиск и фильтры** - мгновенный поиск по тексту, фильтры по статусу, приоритету и категории. Работают совместно.
- **Сортировка** - по дедлайну, приоритету и дате добавления с переключаемым направлением.
- **Цветовая индикация сроков** - зелёный (>24ч), жёлтый (<24ч), красный (просрочено).
- **Боковая панель** - разделы «Сегодня», «Входящие», «Предстоящие», «Завершённые» и проекты (Работа, Учёба, Дом).
- **Модальное окно** - создание и редактирование задач в красивом попапе.
- **Адаптивность** -корректно выглядит на десктопе, планшете и телефоне (mobile-first адаптация).
- **Хранение данных** - задачи сохраняются в LocalStorage и не теряются после перезагрузки.

## 🛠️ Технологии
- HTML5, CSS3 (Flexbox, Grid, кастомные свойства, анимации)
- JavaScript (ES6+)
- Работа с DOM, события, делегирование
- Fetch API, асинхронность (async/await)
- LocalStorage
- Валидация форм (blur + submit)
